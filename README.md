### 2.2. Этап 2: Изучение архитектуры VAE

**Теоретические вопросы:**

| Компонент | Описание |
|-----------|----------|
| **Энкодер** | преобразует входное изображение в параметры распределения в латентном пространстве (μ и log σ²) |
| **Латентное пространство** | вероятностное пространство скрытых признаков |
| **Декодер** | восстанавливает изображение из точки латентного пространства |

**Функция потерь VAE:**

| Компонент | Формула | Назначение |
|-----------|---------|------------|
| Реконструкционная ошибка | `||x - x'||²` (MSE) или `BCE` | качество восстановления изображения |
| KL-дивергенция | `KL(q(z|x) || p(z))` | регуляризация, приближает латентное распределение к нормальному |

**Итоговая функция потерь:**
```
Loss = Reconstruction_Loss + β * KL_Divergence
```

**Выходные артефакты:**
- Краткое описание компонентов VAE в отчёте
- Формулы функции потерь

### 2.4. Этап 4: Реализация вариационного автокодировщика (VAE)

**Задачи:**

| № | Задача |
|---|--------|
| 1 | Реализовать архитектуру VAE (энкодер → μ и log_var → reparameterization → декодер) |
| 2 | Провести обучение |
| 3 | Визуализировать латентное пространство |

**Reparameterization Trick:**

```
z = μ + σ * ε, где ε ~ N(0, I)
```

**Архитектура VAE:**

| Компонент | Слои |
|-----------|------|
| Энкодер | Flatten → Linear(784 → 256) → ReLU → Linear(256 → 64) → ReLU |
| μ и log_var | Linear(64 → latent_dim) |
| Декодер | Linear(latent_dim → 64) → ReLU → Linear(64 → 256) → ReLU → Linear(256 → 784) → Sigmoid |

**Визуализация латентного пространства:**
- 2D-визуализация точек с цветовой кодировкой по классам (цифрам)
- Наблюдение за кластеризацией

**Выходные артефакты:**
- Обученная модель VAE
- 2D-график латентного пространства (классы разными цветами)
- Визуализация реконструкции
- График функции потерь и KL-дивергенции

### 2.5. Этап 5: Сравнение PCA, AE и VAE

**Метрики качества:**

| Метрика | Формула | Назначение |

**Сравнительная таблица:**

| Метод | Размерность латентного пространства | MSE | PSNR | Примечание |
|-------|--------------------------------------|-----|------|-------------|
| PCA | 32 | | | линейное преобразование |
| AE | 32 | | | нелинейное преобразование |
| VAE | 32 | | | вероятностное латентное пространство |

**Выходные артефакты:**
- Сравнительная таблица метрик
- Визуализация реконструкций для PCA, AE, VAE
- Выводы о преимуществах и недостатках каждого метода

| № | Задача |
|---|--------|
| 1 | Сгенерировать изображения из случайных точек латентного пространства VAE |
| 2 | Провести интерполяцию между двумя точками латентного пространства |
| 3 | Исследовать влияние размерности латентного пространства на качество генерации |
| 4 | Добавить шум в латентное пространство и оценить влияние на генерацию |

**Эксперименты:**

| Эксперимент | Параметры | Ожидаемый результат |
|-------------|-----------|----------------------|
| Генерация из случайных точек | z ~ N(0, I) | новые, правдоподобные цифры |
| Интерполяция | линейная интерполяция между z₁ и z₂ | плавный переход между цифрами |
| Влияние размерности | latent_dim = 2, 8, 16, 32, 64 | увеличение dim → лучшее качество |
| Добавление шума | z + ε, ε ~ N(0, σ²) | деградация качества при большом шуме |

**Визуализация интерполяции:**

```
z_start → z_end
(цифра 0)  →  (цифра 9)

Результат: плавное изменение формы, стиля, класса
```

**Выходные артефакты:**

- Графики влияния размерности латентного пространства
- Анализ влияния шума

### 2.7. Этап 7 (опционально): Сравнение VAE и GAN

**Задачи:**

| № | Задача |
|---|--------|
| 1 | Реализовать простую архитектуру GAN (генератор + дискриминатор) |
| 2 | Сравнить качество изображений, сгенерированных VAE и GAN |
| 3 | Провести интерполяцию в латентном пространстве GAN (если возможно) |

**Архитектура GAN:**

| Компонент | Слои |
|-----------|------|
| Генератор | Linear(latent_dim → 256) → ReLU → Linear(256 → 512) → ReLU → Linear(512 → 784) → Tanh |
| Дискриминатор | Linear(784 → 512) → LeakyReLU → Linear(512 → 256) → LeakyReLU → Linear(256 → 1) → Sigmoid |

**Сравнение VAE и GAN:**

| Критерий | VAE | GAN |
|----------|-----|-----|
| Качество отдельных изображений | среднее (размытые) | высокое (резкие) |
| Разнообразие | высокое | может быть низким (mode collapse) |
| Стабильность обучения | стабильное | нестабильное |
| Интерполяция в латентном пространстве | осмысленная | менее осмысленная |

**Выходные артефакты:**
- Сгенерированные изображения от GAN
- Сравнение с VAE
- Выводы о применимости каждого метода
